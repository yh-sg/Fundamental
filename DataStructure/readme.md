# Data Structure
## Object and array
### Object
- Unordered, key value pairs!
- Used when there is no need of any ordering.
- Time Complexity:
    - Insert: O(1)
    - Remove: O(1)
    - Search: O(N)
    - Access: O(1)
- Time Complexity(Methods):
    - Object.keys: O(N)
    - Object.values: O(N)
    - Object.entries: O(N)
    - hasOwnProperty: O(1)

### Array
- Storing of multiple values in a single variable
- Time Complexity:
    - Insert: Depends..
    - Remove: Depends..
    - Search: O(N)
    - Access: O(1)
